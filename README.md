Overview:

This project was created to show my competency using Github and Three.js. It include a tennis court with a ball that simulate a rally being played.

To run the project locally follow the following steps:

1. Clone the project at a desired directory
2. cd to the directory
3. In the terminal run the following command: npm install
4. Lastly run npm run dev, it will create a local link, open the link in the browser to view the website
